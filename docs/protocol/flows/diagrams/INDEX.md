## How to regenerate

```bash
cd ceres-j
python3 tools/render_flow_diagrams.py
```

The script reads every `flows/*.md`, extracts ```` ```mermaid ```` blocks,
sanitizes characters mermaid's parser dislikes (`(` `)` `[` `]`
`;`), and renders each to a PNG via mmdc.

The sanitizer modifies only the on-the-fly tempfile passed to mmdc; the
source markdown stays human-readable. If a diagram fails to render,
inspect the error from mmdc and update the sanitizer in
`tools/render_flow_diagrams.py`.
